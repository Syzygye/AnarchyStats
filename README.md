AnarchyStats is a simple plugin to show server age, total players, and world size in a nice little command. This plugin is mainly focused towards anarchy servers to display their serverâ€™s information.

![Image](https://hyper.lol/wp-content/uploads/2020/11/image.png)

## Features
* See server age, total players, and world size all in one simple command.
* Customize the look and feel of the command.
* Change `/info` to something else.
* Reload the config using `/anarchystats reload`.
